# LDN 呪いのアドオン

LDN 呪いのアドオンは、LDN Server(通称LDNS)で作られたいろいろな呪いのバージョンを参考にして作られたアドオンです。
自称世界一安全にダウンロードできる呪いのアドオンです。(適当)
呪いの内容は様々。変なテクスチャのゾンビが出てきたり、爆弾を持ったゾンビが出てきたり、突然雷が降ってきたり…など
友達へのドッキリ、Minecraft動画のネタ、肝試しなどに使ってみてはいかがでしょうか？
